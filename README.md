# Como rodar

A aplicação foi feita com Java 11

Buildar o projeto 

```
mvn clean compile
```

Rodar suite de testes:

```
 mvn -Dtest=com.trabalhopratico1.AllTests test
```


Rodar todos os testes:

```
mvn test
```
